## Create VENV:

In Git Bash: 

`virtualenv venv`

- then activate: 

`. Scripts/activate`


## IN VSCode

* add venvPath in VScode settings:

`"python.venvPath": "C:\\Users\\USERNAME\\python-projects\\"`

* add Python Interpreter 

- Do CTRL + SHIFT + P
- Python: Select Interpreter
- A list of venv python.exe should show up
